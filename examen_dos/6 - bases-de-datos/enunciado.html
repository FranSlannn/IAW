Bases de datos - Comentarios

  • El directorio actual contiene 3 archivos php
    • home.php (menú principal)
    • crear_base_de_datos.php (Script de creacion de la base de datos 'prueba')
    • crear_tabla.php (Script de creacion de la tabla 'pregunta')
  • Cree los siguientes archivos php con su correspondiente funcionalidad:
    • insertar.php (Insertar un registro en la tabla `pregunta`)
    • actualizar.php (Actualizar un registro en la tabla `pregunta`)
    • eliminar.php (Eliminar un registro de la tabla `pregunta`)
  • Use las siguientes queries
    • INSERT INTO `prueba`.`pregunta` (`name`) VALUES ('test');
    • UPDATE `prueba`.`pregunta` SET `name`=' 22222' WHERE `id`=1;
    • DELETE FROM `prueba`.`pregunta` WHERE `id`=1;
